Parent DMV Checklist

  • Driving school permit application/proof from the school.
  • Accepted identity document for Cameron.
  • Proof of Social Security number.
  • Accepted Illinois residency documents.
  • Parent/legal guardian present or ready to sign where required.
  • Payment method for any fee.
  • Glasses/contacts if he needs them.
  • He has taken multiple random 35-question exams and scored 90% or better.

GitHub Pages upload instructions

  1. Create a free GitHub account.
  2. Create a public repository named illinois-permit-course.
  3. Click Add file → Upload files.
  4. Drag in index.html from this package.
  5. Click Commit changes.
  6. Go to Settings → Pages.
  7. Choose Deploy from branch, branch main, folder /root, then Save.
  8. Wait a minute. Your site will be at https://YOURUSERNAME.github.io/illinois-permit-course/.
