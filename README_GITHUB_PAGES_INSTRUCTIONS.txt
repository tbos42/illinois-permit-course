Cameron's Illinois Permit Prep App

This folder contains a complete static website with:
- index.html

How to publish free on GitHub Pages:
1. Create a free GitHub account.
2. Create a public repository named illinois-permit-course.
3. Open the repository.
4. Click Add file -> Upload files.
5. Upload index.html.
6. Click Commit changes.
7. Go to Settings -> Pages.
8. Under Build and deployment, select Deploy from branch.
9. Select branch: main and folder: /root.
10. Click Save.
11. Wait about 1 minute.
12. Open: https://YOURUSERNAME.github.io/illinois-permit-course/

To update the site later:
1. Return to the repository.
2. Click index.html.
3. Click the pencil/edit icon or upload the replacement index.html.
4. Commit changes.
5. The same website link updates automatically.
